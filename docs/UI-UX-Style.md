# UI / UX Style Guide

视觉与交互的统一规则。任何动画、颜色、文案都按这里走。

## 1. 角色设计

### 比例

- **大头小身**。头与身体高度比约 1.4 : 1，宽度比约 2 : 1。
- 头部为圆形，戴 Hermes 翅膀头盔（覆盖头顶约一半）
- 身体为圆角矩形，明显比头窄
- 小臂、小脚极简（小椭圆 / 小圆角矩形）

### 头盔

- 形状：圆顶矮帽，紧扣头顶
- 翅膀：两只对称的角形翅膀，从太阳穴位置向外伸出
- 头盔正中有一道小高光 / 装饰带（金色色调）

### 翅膀状态（模式指示）

| 模式 | 左翅膀 | 右翅膀 |
|---|---|---|
| 普通模式 | 亮金色 #FFB840 + 三道辐射状细线 | 灰色 #B8B8B8 |
| 工作模式 | 灰色 #B8B8B8 | 亮蓝色 #5DA8E8 + 三道辐射状细线 |

辐射线的实现方式 —— 不用 glow 滤镜，靠"亮色 + 翅膀外缘 3 条细线"营造发光感。

### 表情

- 双眼为黑色椭圆，左上有一颗小白点高光（显得有神）
- 嘴是一段简单弧线（小弧度上翘）
- **状态变种**：
  - 涨红脸（等待 / 接收 Hermes 输出中）：两侧脸颊画圆形粉色 #F5A0B0 半透明圆斑
  - 睡眠 zzz：眼睛变成两条小弧线（闭眼），头顶飘 zzz
  - 摸头反馈：眼睛半眯（眯眼弧线），翅膀微抖

## 2. 色板

| 用途 | 颜色 | Hex |
|---|---|---|
| 身体 / 头主色 | 米色 | #F5E6D3 |
| 身体阴影 / 关节 | 米黄 | #E8DAC5 |
| 头盔主色 | 古铜金 | #C9A77A |
| 头盔描边 | 深棕 | #7A6855 |
| 普通模式翅膀（active） | 金色 | #FFB840 |
| 普通模式翅膀描边 | 暗金 | #A8780C |
| 工作模式翅膀（active） | 浅蓝 | #5DA8E8 |
| 工作模式翅膀描边 | 海蓝 | #1E5A8C |
| 翅膀（inactive） | 灰 | #B8B8B8 / 描边 #888 |
| 眼睛 / 高对比文字 | 黑 | #1A1A1A |
| 涨红脸腮红 | 粉 | #F5A0B0 |
| 气泡背景 | 白 | #FFFFFF |
| 气泡描边（工作模式联动） | 浅蓝 | #5DA8E8 |
| 占位 / 次要文字 | 浅米灰 | #9E9885 |
| 桌面框背景 | 米白 | #F4F1EA |

**色板原则**
- 整体偏暖米调，传"小信使"的温柔感
- 蓝色仅出现在工作模式的右翅膀和气泡描边
- 任何状态都不引入饱和度过高的色（避免变玩具风）

## 3. 字体

- 默认系统中文字体（PingFang SC / 系统默认）
- 气泡内文字 14px regular，placeholder 12px 浅灰
- 不出现粗体堆砌，最多 medium (500)

## 4. 动画原则

### 全局基调

**克制 + 节奏感。** 桌宠有动作，但每个动作都短（< 600ms），不挡住操作。

### 动画清单

| 触发 | 动画 | 时长 | 模式 |
|---|---|---|---|
| 默认（普通模式） | 微微呼吸（身体上下浮 1px） | 持续 | 普通 |
| 每 60s | idle —— 转圈或扭头（二选一随机） | 800ms | 普通 |
| 鼠标长时间不靠近 | 渐变进入 zzz 睡眠（眼变弧线 + 头顶 zzz） | 进入 1.2s，持续 | 普通 |
| 点击头部 | 摸头反馈：眯眼 + 翅膀抖两下 | 500ms | 普通 |
| 拖东西靠近 | 朝鼠标方向倾斜 + 张嘴 | 300ms | 两种模式 |
| 接住成功 | 嘴合上 + 微微一弹 | 200ms | 两种模式 |
| 切换模式 | 普→工：扑棱翅膀飞向右上角；工→普：从右侧走出来 | 600ms | — |
| 等待 / 接收 Hermes 输出中 | 持续涨红脸（脸颊粉色） | 直到流结束 | 工作 |
| 任务完成 | 挥一下右手（露出来那只手或翅膀） | 400ms | 工作 |
| hover 头部（工作模式） | 三个气泡从头部左下方依次淡入弹出 | 200ms（间隔 50ms） | 工作 |
| 离开 hover | 三气泡同时淡出 | 150ms | 工作 |

### 缓动

- 出现 / 进入：ease-out
- 消失 / 离开：ease-in
- 弹跳类（接住、挥手）：cubic-bezier(0.34, 1.56, 0.64, 1)（轻微回弹）

## 5. 工作模式三气泡

### 布局

- 位置：桌宠头部左侧，竖直堆叠
- 顺序（从上到下）：research → 对话 → cowork
- 单个气泡尺寸：宽 170px，高 32px，圆角 16px（药丸形）
- 间距：上下间隔 13px

### 气泡内容

- 左侧：功能名（black, medium）
- 右侧：placeholder（浅灰，如"研究主题…"）
- 对话气泡的功能名右边带 `⌄ system prompt`，点击展开

### 默认显隐

- 工作模式默认仅露半头 + 单侧翅膀
- hover 头部 → 三气泡浮出
- 三气泡浮出后，鼠标离开"头部+气泡"整个区域才收起（避免误触）

## 6. 拖放视觉

- 拖动开始：桌宠"察觉"，头朝鼠标方向偏一点
- 拖动经过桌宠：嘴张开
- 工作模式下，拖动经过任意气泡：该气泡边框变粗 + 高亮
- 松开：嘴合上 + 一次小弹跳；目标气泡进入"装载"状态（占位文本被替换为拖入内容）

## 7. 声音

**默认开启，设置里可关。** 三处轻声，全部低响度（< -20dB）：

| 触发 | 声音 |
|---|---|
| 摸头 | 一声短促的小哼唧（200ms 内） |
| 进入睡眠 | 软软的呼吸声（一次） |
| 任务完成挥手 | 清亮的"叮"（< 300ms） |

其他场景一律静音 —— 切换模式不发声、idle 不发声、hover 不发声。

## 8. 提醒克制度

只在任务完成时挥一下手 —— 不弹通知、不闪烁、不滚动文字。任务结果靠用户主动点开气泡查看。

桌宠绝不主动打断 —— 它不会自己说话、不会冒消息气泡问"需要我吗？"。所有交互都由用户发起。
